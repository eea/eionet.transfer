Welcome to Eionet Web Transfer! This website can facilitate a private transfer of files of up to 10 GB between you and any number of recipients.

The site is only open to uploaders with an Eionet account.

When you have uploaded your file it will show you a URL, which you can email to the intended recipients. The file will only be stored on the website for a limited number of days.
